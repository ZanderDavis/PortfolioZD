Focus Now Overview

Focus Now is a To-Do list app that anyone can use. It will allow users to create multiple lists and enable them to edit, delete, and mark off items whenever they want. It will also allow users to leave notes next to items in the lists, allowing them to make more in-depth notes without worrying about clutter when looking at their list. This program will be for Android users and will be built using Kotlin.

Tools and Languages

  • Languages: Kotlin
